# TicTacToeGame

Tic Tac Toe is a classic two-player game developed as a programming 
project to practice core programming concepts and problem-solving skills.
The game allows two players to take turns placing their
marks (X and O) on a 3x3 board, with the
goal of getting three marks in a row horizontally, vertically, or diagonally.

This project demonstrates concepts such as
game logic implementation, user input handling,
conditional statements, loops, and object-oriented programming principles

# Features
  - friendlly User GUI
  - demonstrates divide and conquer concept
  - use controls evectively and their properties
